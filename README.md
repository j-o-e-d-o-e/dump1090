# flight_tracker

Tracks flights in a specified area, takes photos (for raspberry pi with hq-camera) and sends data to server ([repo](https://github.com/j-o-e-d-o-e/traffic_tracker_api) for Web-API).

## Materials

- Based on `dump1090` by [antirez](https://github.com/antirez/dump1090) and [MalcomRobb](https://github.com/MalcolmRobb/dump1090)
